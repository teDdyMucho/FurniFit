## Tech Stack

- **Frontend**: React 18 + TypeScript
- **Styling**: TailwindCSS with custom glassmorphism
- **Routing**: React Router v6
- **Icons**: Lucide React
- **Build Tool**: Vite

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
npm run build
```

## Design Philosophy

FurniFit uses a futuristic, minimal design with:
- Glassmorphism effects
- Soft gradients
- Clean, breathable spacing
- Mobile-first responsive design
- Modern rounded typography
